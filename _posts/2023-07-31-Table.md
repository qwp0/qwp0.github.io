## 테이블(table)
- 표(table)를 만들 때 사용하는 태그
- tr : 표 내부의 행(table row)
- th : 행 내부의 제목 셀(table heading)
- td : 행 내부의 일반 셀(table data)

## 테이블 태그의 속성

- border
  - 표 테두리 두께 지정
- colspan
  - 열을 합치는 속성. 즉, 해당 셀이 점유하는 열의 수 지정 
- rowspan
  - 행을 합치는 속성. 즉, 해당 셀이 점유하는 행의 수 지정
